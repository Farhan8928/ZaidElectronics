# Zaid Electronics — SEO / GEO / AEO Plan

Goal: when anyone in Mumbai searches "TV repair", "LED/LCD TV repair near me",
"TV panel repair", "screen lines repair", a brand + "TV repair", or asks an AI
assistant, **your Google Business Profile and website (zaidelectronicsmumbai.com)
should show up.**

There are two halves: (A) what's already built into the website, and (B) the
off-site work you (or I) still need to do. Local ranking is ~50% Google Business
Profile + reviews, so **Section B matters as much as the site.**

---

## A. Done on the website ✅

- **Domain**: canonical set to `https://zaidelectronicsmumbai.com/`.
- **Title & meta description**: keyword-rich ("TV repair in Mumbai — LED, LCD &
  Panel Bonding") with brands, free diagnosis, warranty, rating, phone.
- **Structured data (Schema.org JSON-LD)** — validated:
  - `LocalBusiness / ElectronicsStore` — full NAP, geo, hours, services,
    area served, `aggregateRating` 4.9 / 190, founder, payment methods.
  - `FAQPage` — 6 Q&As → feeds Google "People Also Ask", AI Overviews, voice.
  - `WebSite` entity.
- **Geo meta tags** (region IN-MH, Cheeta Camp coordinates) for the local pack.
- **Open Graph + Twitter cards** + a branded share image (`/og-image.jpg`) — link
  previews on WhatsApp / Facebook look professional.
- **robots.txt** — allows Google **and** AI crawlers (GPTBot, PerplexityBot,
  ClaudeBot, Google-Extended, Bingbot…) so answer engines can cite you (GEO).
- **sitemap.xml**, `<noscript>` fallback with NAP + services, semantic headings,
  descriptive image alt text, fast load (Vite, ~95 kB JS gzipped).

> ⚠️ If you use a **www** or different final URL, update it in 4 places:
> `index.html` (canonical/OG/JSON-LD), `public/robots.txt`, `public/sitemap.xml`.

## B. Off-site — do these after launch (highest impact first)

### 1. Google Business Profile (GBP) — your #1 local ranking asset
You already have it (4.9★, 190 reviews). Make it perfect:
- [ ] Claim/verify ownership at business.google.com.
- [ ] **Name**: "Zaid Electronics" (don't stuff keywords into the name — against
      Google rules and risks suspension). Keep the category keywords in the
      *description*, not the name.
- [ ] **Primary category**: "Television repair service". Add secondary:
      "Electronics repair shop", "Appliance repair service".
- [ ] **NAP must match the website exactly** (same spelling of the address/phone).
- [ ] Add **services** (LED TV Repair, LCD TV Repair, Panel Repair & Bonding,
      Smart/Android TV, Doorstep Repair, Buy & Sell) with short descriptions.
- [ ] Set the **service area** (Trombay, Chembur, Govandi, Mankhurd, Cheeta Camp,
      Sion, Kurla) and mark that you do doorstep visits.
- [ ] Upload **real photos**: shop front, workbench, repairs in progress,
      before/after screen-line fixes, the team. 10+ photos; add a few monthly.
- [ ] Add the **website link** to zaidelectronicsmumbai.com.
- [ ] Turn on **WhatsApp/chat** and messaging.
- [ ] Post a **Google Post** weekly (offer, a repair you did, a tip). Free reach.
- [ ] Fill **Q&A** on the profile with the same FAQs as the site.

### 2. Reviews — the strongest trust + ranking signal
- [ ] Ask **every happy customer** for a Google review. Aim for 4–8 new reviews
      per month (steady beats bursts).
- [ ] Make it 1-tap: create a short review link (from GBP → "Ask for reviews")
      and put it in a WhatsApp message template + a QR code sticker at the shop.
- [ ] **Reply to every review** (thank positives; calmly resolve negatives) —
      Google rewards owner responses and it builds trust.
- [ ] Encourage reviewers to mention the **service + area** naturally
      ("fixed my Samsung LED screen lines in Chembur") — those words help ranking.

### 3. Local citations / directories (NAP consistency)
List the business on these with the **identical** name, address, phone, website.
Consistency across the web is what makes Google (and AI) trust the data:
- [ ] Justdial, Sulekha, IndiaMART, AskLaila, TradeIndia
- [ ] Bing Places, Apple Maps (Apple Business Connect)
- [ ] Facebook Page + Instagram (link the site; post repairs)
- [ ] JustDial "verified" + any local Trombay/Chembur directories
- [ ] DoorstepHub (already listed) — update it to match the new NAP + website.

### 4. Search Console + indexing (technical, one-time)
- [ ] Add the site to **Google Search Console**, verify the domain.
- [ ] Submit `https://zaidelectronicsmumbai.com/sitemap.xml`.
- [ ] Use "Request Indexing" on the homepage.
- [ ] Add to **Bing Webmaster Tools** too (Bing powers ChatGPT search).
- [ ] Run the page through Google's **Rich Results Test** and **Schema Validator**
      to confirm the LocalBusiness + FAQ show as eligible.

### 5. AEO / GEO — showing up in ChatGPT, Perplexity, Google AI, voice
The site is already structured for this (FAQ schema, clear Q&A, NAP). To reinforce:
- [ ] Keep hours/phone/address **identical everywhere** — if they conflict across
      GBP/Justdial/site, AI drops you from "open now / near me" answers.
- [ ] The FAQ answers lead with the answer in the first sentence (already done) —
      keep that style if you add more.
- [ ] Reviews mentioning specifics give AI concrete things to cite.

## C2. Multilingual — Hindi / Marathi / Hinglish (how locals actually search)

Most customers type in **Hinglish** ("TV repair wala", "TV theek karne wala",
"LED TV theek karne wala", "ghar par TV repair"), some in **Hindi**
(टीवी रिपेयर, टीवी ठीक करने वाला) and **Marathi** (टीव्ही दुरुस्ती).

**Already added to the site:**
- 66 multilingual keywords in the meta tag (English + Hinglish + Hindi + Marathi).
- The same terms in the JSON-LD `keywords` and `knowsAbout` (read by AI / answer
  engines), plus Hindi & Marathi paragraphs in the `<noscript>` fallback.

**But Google mostly ranks these via your Business Profile, not meta tags** — it
already maps "TV repair wala near me" and Devanagari queries to local businesses.
So to win these searches:
- [ ] In **GBP**, add a Hindi/Marathi line to the business **description**
      (e.g., "टीवी रिपेयर, एलईडी/एलसीडी पैनल रिपेयर, घर पर सर्विस — मुंबई").
- [ ] Encourage some customers to leave **reviews in Hindi/Marathi** ("घर आकर टीवी
      ठीक किया, बढ़िया काम") — these rank you for those exact phrases.
- [ ] On Justdial/Sulekha, many allow a Hindi business description — add one.
- [ ] Do **not** stuff Hindi/Marathi keyword walls onto the visible website —
      Google penalises that. Keep the visible site clean; the schema + GBP +
      reviews carry the multilingual weight.
